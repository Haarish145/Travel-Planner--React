## 5-Minute Setup

### 1. **Start the App**
```bash
cd travel-planner
npm start
```
Opens at `http://localhost:3000` automatically

### 2. **Create Your First Trip**
- Click **"Start Planning"** button
- Fill in the form:
  - **Destination**: Where are you going? (e.g., "Tokyo")
  - **Duration**: How many days? (e.g., "10")
  - **Start Date**: When? (pick a date)
  - **Budget**: How much? (e.g., "3000")
  - **Category**: What type? (pick from dropdown)
  - **Itinerary**: Describe your plans (optional)
  - **Notes**: Any reminders? (optional)
- Click **"Create Trip"**

### 3. **Manage Your Trips**
- **View Details**: Click any trip card to see full information
- **Filter**: Click category buttons to show specific types
- **Sort**: Use dropdown to organize your trips
- **Edit Notes**: Click "Add Notes" in trip details
- **Delete**: Click delete button and confirm

### 4. **All Data Saved**
- Trips automatically save to your browser
- Data persists after closing/reopening
- No account needed!

## Troubleshooting

### "App won't start"
```bash
npm install
npm start
```

### "Lost my trips"
- Check if browser cache was cleared
- localStorage contains your data
- All trips should be there!

### "Form won't submit"
- Fill in all required fields (marked *)
- Duration must be a number
- Budget must be a positive number
- Try refreshing the page
